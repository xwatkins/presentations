What is a style guide?

  • Code standards document
  • Visual language (typography, colours, ...)
  • Sometimes called pattern library

Visual language

Pattern library

Why do we need a style guide?

  • Cleaner code
  • Consistency (code / visual)
  • Time saving
  • Avoid CSS mountain
  • Easier to test (layout)

Some examples

  • NHS Brand guidelines
  • TFL design standards
  • BBC GEL
  • Mozilla style guide
  • Apple Human Interface Guidelines
  • Google Material design

Making our own

  1. Identify elements and components to be added
  2. Start developping base styles for core elements
  3. Add components which override base styles (search boxes, messages, ...)
  4. Add layout and slot components into place

Document code practices: semantic names, ...
